OnlineDelivery

Description

This is a backend Api Designed to manage an online delivery system. The features include Product management,Order Management,User Management And Delivery Management.This is a mvp version without payement integration.

Modules and users

The System can incorporate three levels of users namely the siteadmin,deliverystaff and the common customer.Site admin is responsible for the management of addition updation and deletion of products,managing orders and assigning it to corresponding staff.The delivery staff can keep track of orders to be delivered and the one's already delivered.They hav the permission to update the status of an order after completion of a delivery.The common user can create an account, login to the site sort various products by their categories and place orders.the user can also view placed orders and history of previous orders.

There are four modules for the management of each functionality.

  • Product Management
  • Order Management
  • User Management
  • Delivery Management

URL's

/api/users/Profiles/
Create/update user profiles
/api/users/Login/
User Login
/api/users/Logout/
User Logout
/api/products/CategoryList/
Lists various categories
/api/products/ProductList/
Lists various products
/api/products/ProductList/?cat=id
Lists various products according to the category
/api/orders/OrderList/
Display Orders Placed by the current logged in user
/api/deliveries/DeliveryList/
Display The deliveries assigned to the logged in delivery staff
